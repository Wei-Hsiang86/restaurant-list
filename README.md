# AC exercising _ My Restaurant List

練習製作餐廳清單，可以瀏覽收藏的餐廳、相關資訊等等

## 功能

- 列出所有餐廳
- 瀏覽餐廳相關資訊
- 連結餐廳的地址到 Google 地圖
- 關鍵字查找相關餐廳

## 使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

   ```
   npm install
   ```

4. 安裝完畢後，繼續輸入：

   ```
   npm run start
   ```

5. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

   ```
   Listening on http://localhost:3000
   ```

6. 若想要停止伺服運作

   ```
   ctrl + c
   ```

## 開發工具

- Node.js 18.15.0
- Express 4.18.2
- Express-Handlebars 7.1.2
- Bootstrap 5.1.3
- Font-awesome 6.4.2
